## 🚀 Sobre o Projeto

Este projeto é uma plataforma autoral que serve como meu cartão de visitas no mundo tech. O foco principal foi criar uma interface limpa (**UI**) e uma experiência de uso fluida (**UX**), garantindo que o conteúdo seja acessível em qualquer dispositivo.

### 🛠 Tecnologias Utilizadas

* **HTML5:** Estrutura semântica para melhor SEO e acessibilidade.
* **CSS3:** Estilização moderna utilizando Flexbox/Grid e animações sutis.
* **JavaScript (ES6+):** Manipulação de DOM para interatividade e dinamismo.
* **Design Responsivo:** Adaptado para Mobile, Tablet e Desktop.
